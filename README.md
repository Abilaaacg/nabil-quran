## ✨ المميزات

| الميزة | الوصف |
|--------|--------|
| 📖 **القرآن الكريم** | تصفح جميع سور القرآن الكريم مع التفسير وتحكم في حجم الخط |
| 🎙️ **القراء** | استمع لأكثر من **22 قارئ** من كبار قراء العالم الإسلامي |
| 🕌 **مواقيت الصلاة** | أوقات دقيقة بناءً على موقعك مع عداد تنازلي للصلاة القادمة |
| 📿 **الأذكار** | أذكار الصباح والمساء والصلاة والنوم والطعام والتسبيح |
| 🛡️ **حصن المسلم** | كتاب حصن المسلم كاملاً بتصميم سهل الاستخدام |
| 📻 **الإذاعة** | إذاعات قرآنية مباشرة من مكة المكرمة ومصر وقطر وغيرها |
| 🌙 **وضع الليل/النهار** | تبديل سهل بين الوضعين |
| 📱 **متجاوب** | يعمل على جميع الأجهزة والشاشات |

## 🛠️ التقنيات المستخدمة

```
React 19 + Vite 8      →  واجهة المستخدم
React Router DOM 7     →  التنقل بين الصفحات
Adhan.js               →  حساب مواقيت الصلاة
alquran.cloud API      →  بيانات القرآن الكريم
mp3quran.net           →  تلاوات القرآن الكريم
ipapi.co               →  تحديد الموقع الجغرافي
Netlify                →  الاستضافة والنشر
```

## 🚀 تشغيل المشروع محلياً

```bash
# تحميل المشروع
git clone https://github.com/Abilaaacg/nabil-quran.git
cd nabil-quran

# تثبيت المكتبات
npm install

# تشغيل بيئة التطوير
npm run dev

# بناء النسخة النهائية
npm run build
```
